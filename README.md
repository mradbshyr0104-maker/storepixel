# متجر مقاطعة بكسل

## نشر الموقع على Vercel

### الطريقة 1: عبر الموقع (الأسهل)
1. اذهب إلى [vercel.com](https://vercel.com)
2. سجل دخول بحساب GitHub
3. اضغط "New Project"
4. ارفع المجلد أو اربطه بـ GitHub
5. اضغط "Deploy"

### الطريقة 2: عبر سطر الأوامر
```bash
npm install -g vercel
cd d:\Store.html
vercel
```

## بدائل أخرى

### Netlify
1. اذهب إلى [netlify.com](https://netlify.com)
2. اسحب المجلد وأفلته
3. انتظر النشر

### GitHub Pages
1. أنشئ مستودع GitHub
2. ارفع الملفات
3. فعّل GitHub Pages من الإعدادات
